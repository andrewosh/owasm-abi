# owasm-abi
Derivation macros for traits to bind dispatch and invoke using `owasm-abi`.

[Documentation](https://docs.rs/owasm-abi-derive/)

# Add dependency

```toml
[dependencies]
owasm-abi-derive = "0.1"
```
